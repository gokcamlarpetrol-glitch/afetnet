🎯 GESTURE PARAMETRELERİ:

THRESHOLD (Eşik):
- 50 piksel kaydırma
- Küçük hareketler göz ardı edilir
- Kasıtlı kaydırmalar algılanır

VELOCITY (Hız):
- 500 piksel/saniye
- Hızlı kaydırma daha hassas
- Yavaş kaydırma daha kontrollü

DIRECTION (Yön):
- translationY < -50 → KAPAT (aşağı)
- translationY > 50 → AÇ (yukarı)
- velocityY < -500 → KAPAT (hızlı aşağı)
- velocityY > 500 → AÇ (hızlı yukarı)

⚡ TEKNİK DETAYLAR:

GESTURE HANDLER:
```typescript
import { PanGestureHandler, State } from 'react-native-gesture-handler';

<PanGestureHandler
  onHandlerStateChange={onSwipeGesture}
  onGestureEvent={onSwipeGesture}
>
  <Animated.View>
    <Pressable onPress={toggleControls}>
      {/* Toggle button content */}
    </Pressable>
  </Animated.View>
</PanGestureHandler>
```

SWIPE LOGIC:
```typescript
const onSwipeGesture = (event: any) => {
  const { translationY, velocityY, state } = event.nativeEvent;

  if (state === State.END) {
    const threshold = 50;
    
    if (translationY < -threshold || velocityY < -500) {
      // Aşağı kaydırma → Kapat
      if (isControlsExpanded) toggleControls();
    } else if (translationY > threshold || velocityY > 500) {
      // Yukarı kaydırma → Aç
      if (!isControlsExpanded) toggleControls();
    }
  }
};
```
