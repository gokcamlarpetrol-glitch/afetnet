  🔥 EN MİNİMAL TAB BAR KODU - HİÇBİR ÖZELLEŞTİRME YOK!
═══════════════════════════════════════════════════════════════

YAPILANLAR:

1. TÜM ÖZELLEŞTIRMELER KALDIRILDI ❌
   - screenOptions → SİLİNDİ
   - tabBarStyle → SİLİNDİ
   - tabBarActiveTintColor → SİLİNDİ
   - tabBarLabelStyle → SİLİNDİ
   - tabBarItemStyle → SİLİNDİ
   
2. SADECE TEMEL YAPILDI ✅
   - headerShown: false (her tab'da)
   - tabBarLabel (basit metin)
   - tabBarIcon (basit icon)

3. REACT NAVIGATION DEFAULT KULLANILIYOR
   → Kendi otomatik ayarlarını yapıyor
   → 6 tab eşit şekilde yerleşecek

───────────────────────────────────────────────────────────────
KOD YAPISI:

<Tab.Navigator>  ← ÖZELLEŞTİRME YOK!
  <Tab.Screen 
    name="Home" 
    options={{
      headerShown: false,
      tabBarLabel: 'Ana',
      tabBarIcon: ({ color }) => <Icon />
    }}
  />
  ... (6 tab)
</Tab.Navigator>

───────────────────────────────────────────────────────────────
NEDEN BU ÇALIŞACAK?

React Navigation kendi default stilini kullanacak:
✅ Otomatik genişlik hesaplama
✅ Otomatik renk ayarları
✅ Otomatik padding
✅ Hiçbir çakışma yok
