2. Description of the App

AfetNet is an experimental mobile application that provides offline communication during disaster situations. Main features include:

  • Device-to-device communication via P2P mesh network
  • Experimental early warning system (EEW)
  • Help requests and resource sharing
  • Offline map support
  • Damage report forms
