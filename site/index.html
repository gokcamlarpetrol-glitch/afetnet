About AfetNet

AfetNet is an offline-first disaster response application that enables peer-to-peer communication during emergencies when traditional networks may be unavailable.

Key Features:

  • Offline-First: Works without internet connection
  • P2P Mesh Network: Device-to-device communication via Bluetooth
  • Early Warning System: Experimental earthquake detection (not predictions)
  • Help Requests: Send and receive emergency assistance requests
  • Resource Sharing: Share available resources with the community
  • Offline Maps: Navigate using cached map data

Data Handling:

  • Local Storage: All data stored locally on your device
  • 30-Day Retention: Automatic data cleanup after 30 days
  • Data Export: Export your data in CSV/GeoJSON format
  • Data Deletion: Complete data deletion available in settings
  • Opt-in Telemetry: Anonymous usage statistics only with consent
